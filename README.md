Yêu Cầu Hệ Thống
Phiên bản Python: Đảm bảo bạn đang sử dụng Python 3.11 để đảm bảo khả năng tương thích tốt nhất.
Các Bước Cài Đặt
Thực hiện các lệnh sau trên Terminal hoặc Command Prompt của bạn:
Tải Source Code: Sử dụng Git để sao chép toàn bộ mã nguồn của dự án về máy:
Bash
git clone https://github.com/lamkbvn/schedule-management-web.git
Khởi Chạy Ứng Dụng
Sau khi cài đặt xong các thư viện, bạn có thể khởi động ứng dụng:
Cài đặt Thư viện Phụ thuộc: 
Sử dụng pip để cài đặt tất cả các thư viện cần thiết được liệt kê trong file requirements.txt:pip install -r requirements.txt
Mở Source Code bằng PyCharm:
Mở IDE PyCharm và điều hướng đến thư mục dự án vừa tải về.
Khởi chạy Ứng dụng:
Mở Terminal tích hợp trong PyCharm
Chạy lệnh sau để khởi động máy chủ ứng dụng: python app.py
